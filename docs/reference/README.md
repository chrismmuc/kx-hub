# KX-Hub Reference Documentation

This directory contains technical reference documentation for the KX-Hub MCP server.

## API Reference

- **[MCP Tools API](mcp-tools-api.md)** - Complete reference for all 9 MCP tools
  - `search_kb` - Unified knowledge base search with filtering
  - `get_chunk` - Get chunk details with knowledge cards
  - `get_recent` - Recent reading activity
  - `get_stats` - Knowledge base statistics
  - `list_clusters` - List semantic clusters
  - `get_cluster` - Cluster details with members
  - `configure_kb` - Configuration management
  - `search_within_cluster` - Cluster-scoped search
  - `get_reading_recommendations` - AI-powered reading suggestions
