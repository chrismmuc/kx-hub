## Key Design Decisions (Zusammenfassung)

1. **Filter am Anfang, nicht am Ende** — Sources werden vor dem Generator klassifiziert (nicht aus der privaten Summary extrahiert). Genauer, auch wenn ~2× Compute.

2. **ADK Agent für Filter + Research** — Kein Regelwerk (Allowlist/Denylist/Schwellwerte) für die Relevanz-Entscheidung. Ein ADK Agent auf Vertex AI Agent Engine übernimmt beides: Source-Filterung (kontextuelles Urteil) + Hot News Recherche via `google_search`. Kosten-Delta vs. explizitem LLM-Ansatz: ~$0.25/Monat — akzeptabel für deutlich weniger Wartungsaufwand.

3. **Brevo für E-Mail** — Einziger externer Dienst außerhalb GCP. Free Tier (300/Tag) reicht für den Start. Bei Wachstum auf Brevo Starter ($9/Monat) oder ggf. auf Cloud Run + Listmonk (self-hosted, volle Kontrolle).

4. **Eigener Newsletter ≠ Extrakt der privaten Summary** — Unterschiedliche Pipeline (gefilterter Input, anderer Prompt, englisch, externer Ton). Qualitativ besser als Post-Processing.
